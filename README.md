# ssl_minor_project

## Usage

To run on your local machine

```shell
$ git clone git@github.com:mayankkumrawat/ssl_minor_project.git
$ cd ssl_minor_project/
$ flutter pub get
$ Select a target device
$ Click Run > Start Debugging in the main IDE window, or press F5.
```
